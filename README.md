# Quiz-App

A simple Android Quiz App that allows users to create single- and multiple-choice quizzes. Users can select different categories, difficulty levels, and question types such as multiple-choice or boolean questions.

---

## Features

- Create quizzes with single-choice or multiple-choice questions  
- Select quiz categories (e.g., Science, History, General Knowledge)  
- Choose difficulty levels (Easy, Medium, Hard)  
- Boolean (True/False) question type  
- Track scores and progress  
- Simple, intuitive user interface
